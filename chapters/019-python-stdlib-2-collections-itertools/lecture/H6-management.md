## 5. 자동화 도구 다섯

```bash
black .              # 1. 포매팅
ruff check .         # 2. lint
mypy --strict .      # 3. 타입
pytest -v            # 4. 테스트
pre-commit run --all # 5. 통합
```

자경단 매일 한 줄 의식.

```bash
black . && ruff check . && mypy --strict . && pytest -v
```
