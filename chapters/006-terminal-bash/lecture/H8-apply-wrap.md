## 8. 첫 alias 자신감 — 5분의 시작

본인이 본 챕터를 끝낸 그 5분에:

```bash
# 1. ~/.zshrc 열기 (5초)
$ open ~/.zshrc

# 2. 5종 alias 추가 (1분)
alias s='git status -sb'
alias lg='git log --oneline --graph --all -20'
alias ll='ls -lah'
alias mypr='gh pr list --search "review-requested:@me"'
alias fpush='git push --force-with-lease'

# 3. 저장 + reload (10초)
$ source ~/.zshrc

# 4. 시연 (1분)
$ s
$ ll

# 5. 자축 (3분 호흡)
🎉 본인의 첫 자경단 셸 alias 완성
```

5분 후 본인은 자경단의 진짜 동료. 그 5분이 5년의 시작.

A. 처음 1주일 — deploy·monitor 둘. 1개월 — rollback·backup 추가. 6개월 — migrate.

**Q6. shellcheck CI가 너무 까칠?**
A. 까칠한 게 안전. 1개월 후 본인의 매일 친구.
