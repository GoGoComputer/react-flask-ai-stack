## 1. 5 자료형 깊이

### 1-1. int (정수)

무한대 정수. Python 3에서 long·int 통합.

```python
>>> 5
5
>>> 100_000_000        # 가독성 _
100000000
>>> 0b1010             # binary
10
>>> 0o17               # octal
15
>>> 0x1F               # hex
31
>>> 2**100             # 큰 수 자동
1267650600228229401496703205376
```

### 1-2. float (실수)

IEEE 754 double precision. C의 double과 같음.

```python
>>> 3.14
3.14
>>> 1e10               # 지수
10000000000.0
>>> 0.1 + 0.2          # 함정
0.30000000000000004    # IEEE 754 반올림 오차
>>> from decimal import Decimal
>>> Decimal('0.1') + Decimal('0.2')
Decimal('0.3')         # 정확한 계산
```

자경단 함정 — 화폐 계산엔 Decimal. float 안 됨.

### 1-3. str (문자열)

immutable 시퀀스.

```python
>>> "자경단"
'자경단'
>>> 'single'
'single'
>>> """multi
... line"""
'multi\nline'
>>> "안녕" + " 자경단"  # 연결
'안녕 자경단'
>>> "cat" * 3          # 반복
'catcatcat'
>>> len("자경단")
3
>>> "자경단"[0]         # 인덱싱
'자'
>>> "자경단"[0:2]       # 슬라이싱
'자경'
```

자주 쓰는 메서드 — `.upper()`, `.lower()`, `.strip()`, `.split()`, `.replace()`, `.startswith()`, `.endswith()`, `.find()`.

### 1-4. bool (참/거짓)

`True`·`False` (대문자). int의 subclass.

```python
>>> True
True
>>> False
False
>>> True + 1
2                      # bool은 int의 0/1
>>> bool(0)            # falsy
False
>>> bool("")
False
>>> bool([])
False
>>> bool("text")
True                   # truthy
```

falsy 7종 — `False`·`None`·`0`·`0.0`·`""`·`[]`·`{}`·`()`.

### 1-5. None (없음)

null·nil의 Python 버전. NoneType의 유일 값.

```python
>>> x = None
>>> x is None          # 표준 (== 아님)
True
>>> def foo(): pass
>>> foo()              # 함수 return 없으면 None
>>> print(foo())
None
```

자경단 — `if x is None:` 표준. `if x == None:` ❌ (PEP 8 위반).

## 2. 연산자 26 (자주 쓰는 18)

### 2-1. 산술 7

| 연산자 | 의미 | 예 |
|--------|------|-----|
| `+` | 더하기 | `2 + 3 = 5` |
| `-` | 빼기 | `5 - 2 = 3` |
| `*` | 곱하기 | `4 * 3 = 12` |
| `/` | 실수 나누기 | `7 / 2 = 3.5` |
| `//` | 정수 나누기 | `7 // 2 = 3` |
| `%` | 나머지 | `7 % 2 = 1` |
| `**` | 거듭제곱 | `2 ** 10 = 1024` |

### 2-2. 비교 6

| 연산자 | 의미 |
|--------|------|
| `==` | 값 같음 |
| `!=` | 값 다름 |
| `<` `<=` | 작음·작거나 같음 |
| `>` `>=` | 큼·크거나 같음 |

체이닝 — `1 < x < 10`이 `1 < x and x < 10`과 같음.

### 2-3. 논리 3

| 연산자 | 의미 |
|--------|------|
| `and` | 둘 다 True |
| `or` | 둘 중 하나 True |
| `not` | 반전 |

short-circuit — `False and X`은 X 평가 안 함. `True or X`도 X 안 함.

### 2-4. 할당 8

| 연산자 | 같음 |
|--------|------|
| `=` | 할당 |
| `+=` | `x = x + y` |
| `-=` | `x = x - y` |
| `*=` | `x = x * y` |
| `/=` | `x = x / y` |
| `//=` | `x = x // y` |
| `%=` | `x = x % y` |
| `**=` | `x = x ** y` |

### 2-5. 멤버십 2

| 연산자 | 의미 |
|--------|------|
| `in` | 포함 여부 |
| `not in` | 미포함 |

```python
>>> "까미" in ["까미", "노랭이"]
True
>>> "본인" not in ["까미", "노랭이"]
True
```

자경단 — `if cat in cats:` 매일 사용.

## 3. String Formatting 3종

### 3-1. % (옛, 1990s)

C의 printf 양식. 권장 X.

```python
>>> "안녕 %s, %d살" % ("까미", 3)
'안녕 까미, 3살'
```

### 3-2. .format() (중간, 2008+)

```python
>>> "안녕 {}, {}살".format("까미", 3)
'안녕 까미, 3살'
>>> "안녕 {name}, {age}살".format(name="까미", age=3)
'안녕 까미, 3살'
```

### 3-3. f-string (표준, 3.6+)

```python
>>> name = "까미"; age = 3
>>> f"안녕 {name}, {age}살"
'안녕 까미, 3살'
>>> f"{name=}"           # 디버그 (3.8+)
"name='까미'"
>>> f"{age:03d}"         # 형식 지정
'003'
>>> f"{3.14159:.2f}"
'3.14'
```

자경단 표준 — **f-string**. 가장 빠름·짧음.

## 4. mutable vs immutable

### 4-1. 5 자료형 분류

| 자료형 | mutable? |
|--------|---------|
| int·float·str·bool·None | immutable |
| tuple·frozenset | immutable |
| list·dict·set | mutable |

### 4-2. mutable 함정 5개

```python
# 함정 1: 같은 list 공유
>>> a = [1, 2, 3]
>>> b = a               # b는 a의 별칭
>>> b.append(4)
>>> a
[1, 2, 3, 4]            # a도 바뀜!

# 처방
>>> b = a.copy()        # 또는 b = list(a)

# 함정 2: 함수 default 인자
>>> def add_cat(cat, cats=[]):  # 위험!
...     cats.append(cat)
...     return cats
>>> add_cat("까미")
['까미']
>>> add_cat("노랭이")
['까미', '노랭이']        # 누적됨!

# 처방
>>> def add_cat(cat, cats=None):
...     if cats is None:
...         cats = []
...     cats.append(cat)
...     return cats
```

### 4-3. == vs is

```python
>>> a = [1, 2, 3]
>>> b = [1, 2, 3]
>>> a == b              # 값 같음
True
>>> a is b              # 객체 같음
False
>>> a is a
True
```

자경단 표준 — 값 비교는 `==`, None 비교는 `is None`.

### 4-4. type() vs isinstance()

```python
>>> type(5) == int
True
>>> isinstance(5, int)
True
>>> isinstance(True, int)   # bool은 int의 subclass
True                          # type() == int은 False
```

자경단 — `isinstance` 권장 (상속 처리).

## 7. 흔한 오해 5가지

**오해 5: "mutable이 위험만."** — 효율 + 편의. 함정만 알면 안전.

## 8. FAQ 5가지

**Q1. int의 최대값?**
A. 무한 (메모리만큼). C와 다름. `2**1000`도 OK.

**Q2. float 정확도?**
A. IEEE 754 double = 15~17자리 십진수. 화폐는 Decimal.

**Q3. str의 immutable이 왜?**
A. 안전 + 해시 가능 + dict key. 변경은 새 str 생성.

**Q4. None이 falsy인가요?**
A. Yes. `if x:`은 None·0·""·[] 다 False.

**Q5. == vs is의 면접 답?**
A. "== 값 비교, is 객체 비교. `5 == 5.0` True, `5 is 5.0` False. None 비교는 항상 `is None`".
